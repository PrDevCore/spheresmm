# SphereSMM SaaS Dashboard

React, Express, Firebase, and Gemini-powered social media management dashboard with user-configured social publishing credentials.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set your Gemini and Firebase configuration.
3. Run the app:
   `npm run dev`
